IO

  • Up to /Examples/Core
  • Read.asm.txt Demonstrate how to read input using a trap instruction. Set trap code, address of buffer, and number of characters to read.
  • ReadWrite.asm.txt Read a string and write it. Illustrate both input and output.
  • Write.asm.txt Demonstrate output using trap. Set trap code, buffer address, and size in registers.
  • WriteChunks.asm.txt Write several short strings on the same output line. Trap write doesn't terminate line of output unless \n character is given.
