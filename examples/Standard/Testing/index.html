Testing

  • Up to /Examples/Standard
  • AsmStd.asm.txt Test assembly of each statement. For checking that assembler generates correct object code.
  • BitField.asm.txt Test instructions that operate on bit fields. Examples of extract and extracti.
  • Branch.asm.txt Test the pc relative branch instructions. The numbers in comments indicate expected order of execution.
  • Logic.asm.txt Test the logic instructions and pseudoinstructions. Instructions logicw, logicb. Pseudoinstructions for common operators.
  • Relocatable.asm.txt Test assembly of relocatable and nonrelocatable addresses. Contains some names that are relocatable, others that are fixed.
  • SaveRestore.asm.txt Test save and restore instructions. These instructions simplify procedure call and return.
  • Shift.asm.txt Test the shift instructions. Expected results are given in comments.
  • Stack.asm.txt Test the stack instructions: push, pop, top. Put test data in registers, then perform stack operations.
  • StdArithmetic.asm.txt Test cases for arithmetic. Add with carry, multiply and divide on natural numbers.
  • System.asm.txt Test system control registers and instructions. Examples of putctl, getctl.
