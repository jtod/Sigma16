Arithmetic

  • Up to /../Examples/Core
  • AddDecimal.asm.txt Add BCD (Binary Coded Decimal) numbers. Illustrate BCD representation and arithmetic.
  • BigPowers2.asm.txt Print powers of 2 using BCD arithmetic. Integers of unbounded size are represented as an array of digits.
  • BinToDec.asm.txt Convert binary to decimal. Illustrate how to print a binary number as a string of decimal digits.
  • Div.asm.txt Example of division. Calculate both quotient (in destination register) and remainder (R15).
  • Mult.asm.txt demonstrate the multiplication instruction The mul instruction multiplies signed integers; overflow is possible.
  • Overflow.asm.txt Test detection of overflow. Overflow in an arithmetic operation is indicated in the condition code.
  • WriteBCD.asm.txt Print a BCD number. Illustrate output of an array of decimal digits.
