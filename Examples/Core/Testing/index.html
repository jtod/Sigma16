Testing

  • Up to /Examples/Core
  • Arithmetic.asm.txt Test cases for core arithmetic instructions. Expected results are given in comments.
  • AsmCore.asm.txt Test assembly of each statement in the core architecture. Check that the assembler generates correct object code.
  • AsmErrMessages.asm.txt Test assembly language error messages. This program contains as many errors as possible, on purpose!
  • Cmp.asm.txt Test condition codes set by the cmp instruction Perform a variety of instructions that set condition code bits.
  • Jumps.asm.txt Test Sigma16 Core jumping instructions. Test jump, conditional jumps, and conditional jump pseudoinstructions.
  • Loads.asm.txt test load and lea Sigma16
  • Looper.asm.txt A long-running program for timing measurements. Run loop for n*m iterations, so it can execute a lot of instructions.
  • Memory.asm.txt Test load, store, addresses. Expected results given in comments.
  • TrapBreak.asm.txt Test case for trap break. The program contains a long-running loop, containing a trap break.
